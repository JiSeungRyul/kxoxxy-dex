\# AGENTS.md



\## Project

KxoxxyDex is a Pokemon encyclopedia web project.



\## Stack

\- Next.js App Router

\- TypeScript

\- Tailwind CSS



\## Rules

\- Use clean, production-ready code

\- Prioritize extensibility

\- Use National Dex Number only

\- Do not implement regional dex numbers

\- Do not implement detail page yet

\- Do not implement login/community yet



\## Data

\- Use PokeAPI

\- Prefer reusable data mapping utilities

\- Keep API integration logic separate from UI components



\## UI

\- Desktop-first

\- Clear table/list readability

\- Keep filters and sorting controls modular
