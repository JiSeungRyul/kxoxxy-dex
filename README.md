# KxoxxyDex

KxoxxyDex is a Pokemon encyclopedia web project.

Goal:
Build a modern Pokemon information hub that includes:

* Pokedex
* Pokemon game information
* Pokemon community features

Current Stage (MVP):
Pokedex list page with:

* Name search
* Type filter
* Generation filter
* Stat sorting

Future Plans:

* Pokemon detail pages
* Community discussion
* Pokemon game information archive



Project name: KxoxxyDex

Repository name: kxoxxy-dex

kxoxxydex.com

## MVP Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- PokeAPI

## MVP Scope

- Desktop-first Pokedex list page
- Search by Pokemon name
- Filter by type
- Filter by generation
- Sort by National Dex Number, Name, and base stats
- Ascending and descending sort order
- Reset filters
